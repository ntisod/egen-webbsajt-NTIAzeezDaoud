Miniräknare Med Eval

Projektet gick ut på att göra en miniräknare. Min lösning var att göra en miniräknare genom att ta en sträng som byggs genom att användaren matar in data genom miniräknarens knappar. Därefter så förmatteras strängen till sträng som JavaScript ska kunna beräkna och svaret beräknas genom eval() funktionen som är inbyggd i JavaScript som kör sträng som det vore vara JavaScript kod.
